Debug Library Version Loss

Инструкции для отладки:

  1. Откройте консоль разработчика (F12)
  2. Загрузите плагин в Figma
  3. Запустите проверку компонентов
  4. Следите за логами в консоли с префиксами:
    • [updateAvailabilityCheck] - проверка версий компонентов
    • [updateQueue] - обработка очереди обновлений
    • [index.ts] - отправка данных в UI
    • [UI] - получение данных в UI
    • [UI Group Item] - отображение элементов в группах

Что искать в логах:

  • Наличие libraryComponentVersion и libraryComponentVersionMinimal на каждом этапе
  • Где именно эти данные теряются или становятся undefined/null
  • Правильность передачи данных между JS и UI
